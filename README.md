# Video Language Translator

A web application that translates short-form videos from Chinese to multiple languages (English, Korean, and Japanese) while preserving the speaker's tone and style.

## Features

- Video upload support for Chinese short-form content
- Language selection (English, Korean, Japanese)
- Tone-preserving translation
- Real-time video processing
- Modern, responsive UI

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- FFmpeg for video processing
- Translation APIs
- Voice synthesis APIs

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
